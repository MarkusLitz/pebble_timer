Pebble timer

Timer for the Pebble Smart Watch

View on GitHub

How to install

You can install this timer by downloading the watch app and then following the instructions under 1.2 Install a pre-compiled app from your computer (here.

How to Use

This timer will allow you to quickly setup a single timer. Upon opening the timer you can increment or decrement the timer by pressing and holding the up and down buttons. To start the timer just press the select button.
